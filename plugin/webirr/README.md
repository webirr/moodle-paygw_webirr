# WeBirr Moodle Payment Gateway

This directory is the actual Moodle plugin package root.

To install from source, copy this folder into Moodle as:

```text
payment/gateway/webirr
```

The repository root contains screenshots, example apps, and release notes. This
plugin folder should be the only source used when building the Moodle release ZIP
whose top-level folder must be named `webirr`.
