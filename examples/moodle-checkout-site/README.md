# Moodle Checkout Example Site

This directory is reserved for a real Moodle example environment that installs
and exercises the actual plugin from `../../plugin/webirr`.

Purpose:

- Run Moodle with the WeBirr payment gateway installed.
- Configure a test payment account.
- Create a sample paid enrollment or checkout item.
- Validate the full Moodle checkout flow with TestEnv credentials.
- Capture screenshots from the real Moodle plugin UI.

Status: planned. The current local Docker validation environment exists outside
the repository. The next step is to codify it here with Docker configuration,
seed data, and repeatable screenshot commands.
